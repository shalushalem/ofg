# OFG Connects Local Build

This workspace contains a Flutter Android app and a local Python + SQLite backend.

## Run The Local Backend

```bat
run_backend.bat
```

Backend URL:

```text
http://127.0.0.1:8787
```

Android emulator URL:

```text
http://10.0.2.2:8787
```

Real phone URL:

```text
http://YOUR_PC_LAN_IP:8787
```

Demo login:

```text
demo@ofg.local
password123
```

The SQLite database is created at `backend/ofg_local.db`. Uploaded local media is stored in `backend/media/`.
